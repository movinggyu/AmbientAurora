# AmbientAurora: 프로젝트 계획 및 기획서

## 1. 프로젝트 개요
- **프로젝트 명:** AmbientAurora
- **핵심 아이디어:** PC 모니터를 주변 환경과 사용자 기분에 맞춰 색상을 동적으로 변화시키는 스마트 무드등으로 활용하는 데스크톱 애플리케이션.
- **목표:** 사용자에게 몰입감 높은 시각적 경험을 제공하고, 생산성 향상 및 휴식에 도움을 주는 유틸리티 제작.

## 2. 주요 기능 (Features)

### 2.1. 다채로운 조명 모드
- **단색 모드 (Static):** 사용자가 지정한 단일 색상으로 화면을 채움.
- **그라데이션 모드 (Dynamic Gradient):** 두 개 이상의 색상이 부드럽게 전환.
- **오로라 모드 (Random Aurora):** 실제 오로라처럼 유기적이고 자연스러운 색상 변화 연출.
- **커스텀 모드 (Preset & Timeline):** 사용자가 직접 조명 시퀀스를 디자인하고 저장.

### 2.2. 직관적인 UI/UX
- **미니멀리즘 디자인:** 조명 효과에 온전히 집중할 수 있도록 UI 요소를 최소화.
- **레이어형 오버레이 UI:** 필요할 때만 나타나는 컨트롤 패널.
- **마우스 커서 자동 숨김:** 일정 시간 움직임이 없으면 커서 자동 숨김.
- **도킹 시스템:** 각 기능 패널(슬라이더, 카메라 등)을 메인 패널에 붙이거나 떼어내어 자유롭게 배치.

### 2.3. 고급 컨트롤
- **HSL 색상 모델:** 인간의 직관에 더 잘 맞는 HSL(색상, 채도, 밝기) 슬라이더를 통해 색상 제어.
- **속도 및 변화폭 조절:** 동적 모드에서 색상 전환 속도, 변화의 폭 등을 세밀하게 조절.

### 2.4. 카메라 연동 ('셀카 스튜디오')
- **화면 조명 활용:** 모니터 자체를 조명으로 활용하여 고품질의 셀카 촬영 지원.
- **부가 기능:** 화면 플래시, 실시간 좌우 반전, 타이머, 다양한 해상도 및 비율 설정.

## 3. 상세 설계 (Detailed Design)

### 3.1. UI/UX 상세
- **최소화 상태:** 화면 중앙에 반투명(30%) 로고만 표시. 마우스 호버 시 100% 불투명.
- **메인 컨트롤 패널:** 로고 클릭 시 페이드인 효과와 함께 등장. 프로그램 정보, 현재 상태 표시 및 하위 패널(슬라이더, 카메라 등)을 열 수 있는 버튼 포함.
- **슬라이더/카메라 패널:** 메인 패널 좌우에 자석처럼 붙거나(Docking), 별도 창으로 분리(Pop-out) 가능.

### 3.2. 조명 모드별 알고리즘
- **단색 모드:**
  - 소프트 웜/블루라이트 차단 필터 토글 기능.
- **그라데이션 모드:**
  - 전환 효과 선택: 전체 화면 전환(Fade), 방향성 전환(Linear Push), 중앙에서 퍼짐(Radial Pulse).
  - 사용자가 2개 이상의 색상 정점(Color Stop)을 지정하여 그라데이션 범위 제한.
- **오로라 모드:**
  - **펄린 노이즈(Perlin Noise)** 알고리즘을 사용해 구름이나 연기처럼 자연스럽고 유기적인 색상 흐름 구현.
  - 제어 항목: 변화 주기(Speed), 색상 변동 폭(Deviation), 특정 색상 유지 시간(Dwell Time).
- **커스텀 모드:**
  - 타임라인 기반으로 조명 시퀀스(예: 10초간 붉은색 -> 5초간 푸른색으로 디졸브)를 생성.
  - 생성된 시퀀스를 `.json` 형식의 파일로 저장(Export) 및 불러오기(Import).

### 3.3. 컨트롤러 상세 (HSL 기반)
| 슬라이더 | 제어 범위 | 설명 |
|---|---|---|
| **색상 (Hue)** | $0^\circ \sim 360^\circ$ | 무지개 스펙트럼에서 기본 색상 선택 |
| **채도 (Saturation)** | $0\% \sim 100\%$ | 색상의 선명도 조절 (파스텔톤 ↔ 원색) |
| **밝기 (Lightness)** | $0\% \sim 100\%$ | 색상의 밝기 조절 (검정 ↔ 순색 ↔ 흰색) |
| **속도 (Speed)** | 0.1초 ~ 60초 | 동적/오로라 모드에서 색상 전환 주기 |

### 3.4. 카메라 패널 상세
- **화면 플래시:** 촬영 순간, 화면 전체가 밝은 흰색으로 터져 조명 역할 수행.
- **실시간 좌우 반전 (Mirror Mode):** 거울을 보는 것처럼 구도 설정.
- **타이머:** 3초, 5초, 10초 카운트다운.
- **해상도 및 비율:** $1:1$, $4:3$, $16:9$ 등 SNS 및 표준 규격에 맞는 비율로 크롭(Crop) 기능 제공.

## 4. 기술 스택 및 아키텍처
- **주요 언어:** C++ (C++20 이상 권장)
- **빌드 시스템:** CMake
- **핵심 라이브러리 (제안):**
  - **UI 및 그래픽스:** **Qt** 또는 **Dear ImGui** + **SFML/OpenGL**. 그래픽 효과 및 렌더링에 유연하며, 크로스플랫폼을 지원하는 조합.
  - **카메라 제어:** **OpenCV**. 웹캠 제어 및 이미지 처리에 필수적.
  - **데이터 직렬화 (프리셋):** **nlohmann/json**. C++에서 사용하기 간편한 JSON 라이브러리.
